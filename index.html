3 modules Apache-2.0 licensed Signed .modl builds

Installing a module

Download the .modl file, then in your Ignition Gateway go to Config → Modules → Install or Upgrade a Module and select the file. All builds are signed with the Gaskony certificate.

# Gateway → Config → Modules → Install or Upgrade → select .modl → accept certificate

Questions, bugs, or feature requests?

These modules are open source and actively maintained. The fastest way to get help or shape what comes next is on GitHub — open an issue on the relevant module and it'll be picked up there. Pull requests are welcome too.
