# Gaskony Ignition Modules

Independent, open-source modules for Inductive Automation's
[Ignition](https://inductiveautomation.com/) platform (8.3+).

### ➡️ Download portal: https://gaskony-ignition.github.io/ignition-modules-portal/

| Module | Latest | Source |
| ------ | ------ | ------ |
| Camera Driver | v3.1.9 | [ignition-module-camera-driver](https://github.com/Gaskony-Ignition/ignition-module-camera-driver) |
| Logix PLC Emulator | v10.1.0 | [ignition-module-plc-emulator](https://github.com/Gaskony-Ignition/ignition-module-plc-emulator) |
| Python 3 Integration | v4.5.3 | [ignition-module-python3](https://github.com/Gaskony-Ignition/ignition-module-python3) |

All modules are signed and built for Ignition 8.3+ (Java 17). Download the `.modl`
from a [release](../../releases) and install via
**Gateway → Config → Modules → Install or Upgrade a Module**.

## Questions & contributions

Open an issue on the relevant module's source repository. All modules are
licensed under Apache-2.0 and contributions are welcome.
